
**What is a Network? **

* A set of computers **sharing resources** located on or provided by **network nodes**.
* Host: 
* **DNS (Domain Name System)** is a service that *translates* human‑readable domain names *like google.com* into *the numerical IP addresses* that computers use to locate and communicate with servers on the internet. When you type google.com in your browser, DNS looks up and returns Google’s IP address so your device can connect to the correct server. 
* IP : Internet protocol
* A **port number** is a way to identify a specific process to which a network message is to be forwarded when it arrives at a server
* **NIC (Network Interface Controller)** also known as a **Network Interface Card**, **Network adapter** **Lan adapter** or **Physical Network Interface**
* **MAC (Media Access Control) Address** is an addresses written on a chip of every NIC
* Every device should have a Mac Addresses

**Physical network:** the actual hardware: cables, switches, routers, servers — the tangible infrastructure.
**Logical network:** the virtual structure built on top of the hardware: IP subnets, VLANs, routing rules, and how data logically flows.
